Total Tweet Count

The total sum count of all hashtags is displayed below. This visualization represents the number of times each particular hashtag was tweeted within a certain period of time. Due to limitations in the Twitter API, we are not able to query against all tweets on Twitter. We are limited to very defined requests and specifications/quantities. As such, you will notice that the chart below has equal proportions. If the Twitter API allowed us to query from the entire Twitter database, we would expect these proprtions to reflect the actual number of tweets containing each hashtag. The second chart shows sample data as we would expect to see with an open API.
